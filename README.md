# Дашборд менеджмента товаров в розничной торговле

## Описание проекта

Данный проект представляет собой интерактивный дашборд, разработанный с использованием библиотеки **Dash** на языке программирования Python. Дашборд предназначен для анализа ключевых показателей процесса менеджмента товаров в розничной торговле и позволяет визуализировать финансовые и операционные данные, связанные с управлением ассортиментом.

Проект выполнен в рамках лабораторной работы по теме построения интерактивных дашбордов и демонстрирует базовые принципы работы с визуализацией данных, фильтрацией и пользовательским вводом.

## Структура дашборда

Дашборд состоит из нескольких логически связанных блоков, каждый из которых отвечает за отображение определённых показателей процесса менеджмента товаров.

В верхней части дашборда расположен заголовок и краткое описание, поясняющее назначение приложения и тип анализируемых данных.

Ниже размещён элемент загрузки данных, позволяющий пользователю загрузить CSV-файл с информацией о товарах, продажах, расходах и складских показателях. Данные загружаются через веб-интерфейс и используются для построения всех визуализаций.

После загрузки данных пользователю становятся доступны элементы управления фильтрацией. Выпадающий список категорий позволяет выбрать конкретную товарную категорию для анализа. Выпадающий список периода анализа используется для агрегации данных по месяцам, кварталам или годам.

---

## Описание графиков и элементов

В дашборде реализованы следующие визуальные элементы:

Линейный график динамики доходов и расходов отображает изменение выручки и расходов во времени за выбранный период. Данный график позволяет оценить общую финансовую динамику и выявить тренды.

Круговая диаграмма структуры расходов используется для анализа распределения расходов по периодам или категориям. Она позволяет наглядно оценить, какие затраты формируют основную часть расходов.

Гистограмма распределения прибыли показывает частотное распределение значений прибыли и позволяет оценить стабильность финансовых результатов.

График рассеяния используется для анализа корреляции между прибылью и объёмом продаж. Он позволяет выявить зависимость между ключевыми показателями эффективности.

Таблица данных отображает агрегированные значения финансовых и операционных показателей за выбранный период. Таблица используется для детального просмотра исходных данных и результатов агрегации.

Все графики являются интерактивными и автоматически обновляются при изменении выбранных параметров фильтрации или периода анализа.

## Формат входных данных

Для корректной работы дашборда CSV-файл должен содержать следующие столбцы:

* date — дата
* category — категория товара
* revenue — выручка
* expenses — расходы
* profit — прибыль
* sales_volume — объём продаж
* stock_level — уровень складских остатков

Данные в столбце date должны быть представлены в формате даты.

## Запуск приложения

Для запуска дашборда необходимо установить зависимости и выполнить файл приложения.

Установка зависимостей:

```
pip install dash pandas plotly
```

Запуск приложения:

```
python app.py
```

После запуска дашборд будет доступен в браузере по адресу:

```
http://127.0.0.1:8050
```

## Назначение проекта

Проект предназначен для демонстрации возможностей библиотеки Dash при создании интерактивных дашбордов, а также для анализа данных, связанных с процессом менеджмента товаров в розничной торговле. Полученные результаты могут быть использованы для оценки эффективности управления ассортиментом и поддержки управленческих решений.
